# e-Stat 日本語質問サンプル

e-Stat MCP を injection-tool から試すときの日本語質問サンプル集です。

最初から複雑な依頼を投げるより、まずは `人口` `GDP` `CPI` `失業率` などの統計名、`東京` `全国` などの地域、`2024年` などの時期を入れた方が安定します。

## 使い方のコツ

- まずは `人口` `CPI` `GDP` のような統計名を入れる
- 地域を絞るときは `東京` `大阪` `全国` `都道府県別` を入れる
- 時期を絞るときは `2024年` `最新` `直近` を入れる
- 値を見たいときは `教えて` `見せて` `一覧` `推移` `比較` を入れる
- 表の構造を確認したいときは `項目` `分類` `時間軸` `地域コード` を入れる

## 基本テスト

| やりたいこと | 質問サンプル |
|---|---|
| 人口統計を試したい | 日本の人口統計を教えて |
| CPI を試したい | 東京の最新CPIを教えて |
| GDP を試したい | 日本のGDPに関する統計を探して |
| 失業率を試したい | 日本の最新の失業率を教えて |
| 統計表検索を試したい | 消費者物価指数の統計表を探して |
| 労働統計を試したい | 労働力調査の統計を探して |

## 避けた方がよい曖昧表現

- `統計を見せて`
- `何かデータある?`
- `最近の数字を教えて`
- `これ調べて`
- `日本の状況を教えて`

このレベルまで曖昧だと、どの統計表を探すべきか決めにくく、一般回答に寄りやすくなります。

## 安定しやすい書き方

### 良い例

- 日本の人口統計を教えて
- 東京の最新CPIを教えて
- 都道府県別の人口を見せて
- 日本の失業率の推移を教えて
- 消費者物価指数の統計表を探して
- この統計表の地域項目を教えて

### さらに安定させる例

- 東京の2024年の人口データを見せて
- 全国の最新CPIを教えて
- 2023年と2024年の東京のCPIを比較して
- 都道府県別の人口データを一覧で見せて

## おすすめテスト順

1. `日本の人口統計を教えて`
2. `東京の最新CPIを教えて`
3. `消費者物価指数の統計表を探して`
4. `その統計表の時間軸と地域項目を教えて`
5. `その統計表で東京の2024年データを見せて`
6. `東京と大阪を比較して`

この順で試すと、`search_statistics` → `get_statistic_meta` → `get_statistic_data` の流れが一通り確認しやすいです。
